COVID Parcel Business Analysis

Overview
This project analyzes the impact of COVID-19 on ABC Company's parcel delivery business using Python and data visualization.

Objectives
- Analyze parcel volume trends before and during COVID
- Identify when the impact began
- Evaluate peak season performance
- Assess customer growth and segmentation
- Estimate revenue impact

Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Excel (for verification)
- PowerPoint (executive presentation)

Key Insights
- COVID impact began around Week 12 (2020)
- Parcel volume increased significantly in 2020
- Growth was driven by existing customers
- Revenue increased due to higher shipment volume

files
- Notebook: Data analysis and code
- PowerPoint: Executive presentation
- Excel file for Anaysis: To verify the Python Analytical results 
- CSV files: Raw Data & Processed outputs
